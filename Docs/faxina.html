Objetivo

Este script tem como objetivo realizar a etapa de faxina de dados, ou seja, aplicar transformaes, padronizaes e correes necessrias para garantir a qualidade dos dados utilizados nas anlises. Aqui so tratados problemas como valores ausentes, formatao inconsistente, e possveis outliers.

  mutate(
    data_modif = data -min(data),
    xco2_est = a_co2+b_co2*data_modif,
    delta = xco2_est-xco2,
    xco2_detrend = as.numeric((a_co2-delta) - (mean(xco2) - a_co2))
  )
